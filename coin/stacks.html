Description:

Built on Bitcoin: The Stacks token extends Bitcoin with new functionality instead of starting from scratch. Stacks use Bitcoin as a secure and robust base layer where all transactions are settled. While Bitcoin is helping there, Stacks adds complex apps and smart contracts. Apps built on Stacks can interact with Bitcoin state, so you can have an app that uses Bitcoin as its currency. Clarity Smart Contracts: The logic behind decentralized apps is a smart contract that is a code that runs on the blockchain. Since nobody owns the logic behind decentralized apps, there is no recourse for users if something goes wrong. That’s why Clarity is designed to prevent bugs, common exploits, and to proactively protect the user. Proof of Transfer: Proof of Transfer is the consensus mechanism; introduced by Stacks, that runs between Bitcoin and Stacks. It recycles Bitcoin’s Proof of Work consensus mechanism to accomplish both high levels of decentralization and scalability without additional energy consumption. Proof of Transfer’s other half is called Stacking, which enables holders of Stacks Token to earn Bitcoin.
